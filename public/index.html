Realtime Programming Game

Your World

How to play

Goal: reach green circles by issuing one action per step. The server applies only the first action emitted by your code each tick.

  • You must set your player name first. You cannot change it later.
  • Time limit: each round is 2 minutes.
  • Scoring: success +1 per valid step, +20 per goal, plus bonuses for speed/efficiency. Errors penalized.

Action output per language

JavaScript: Call one of: api.moveForward(), api.turnLeft(), api.turnRight(). Use api.log() to print logs.

Python: Print exactly one token to stdout: MOVE | LEFT | RIGHT.

C: Print exactly one token to stdout: MOVE | LEFT | RIGHT. Your main() should printf a single word.

Examples

JavaScript: move in a square
// Move 3 steps, then turn right — repeat
for (let i = 0; i < 3; i++) {
    api.moveForward();
}
api.turnRight();
Python: step forward each tick
# Print one action per run
print('MOVE')
C: step forward each tick
#include <stdio.h>
int main(){ printf("MOVE\n"); return 0; }
